# AI-Based Loan Repayment Prediction System

This project uses machine learning to predict whether a borrower will repay a loan based on financial and demographic data.

## Features
- Data preprocessing (handling missing values)
- Encoding categorical variables
- Logistic Regression model training
- Model evaluation (accuracy, confusion matrix)
- Individual prediction example

## Tools Used
- Python
- Pandas
- Scikit-learn

## Result
- Achieved ~87% accuracy

## How to Run

1. Place dataset file in the same folder:
   loan_dataset_20000.csv

2. Run:
   python loan_prediction.py

## Note
Dataset was obtained from Kaggle.
